  Gamification, Engagement & Personalization

GOAL:
  Enhance user engagement through gamification and personalization.
  Implemented daily goals, dynamic streaks, achievements, customizable
  timers, study subjects, a leaderboard, and a dedicated settings menu.

===================================================================
  BACKEND UPDATES
===================================================================

1. GOALS & STREAKS API
   ├── File: backend/app/api/goals.py                [NEW]
   ├── GET  /api/goals        → gets user's daily goal + current streak
   ├── PUT  /api/goals        → updates daily goal target
   ├── GET  /api/goals/today  → today's progress vs target
   └── MongoDB collection: "user_goals"

2. ACHIEVEMENTS API
   ├── File: backend/app/api/achievements.py         [NEW]
   ├── GET  /api/achievements      → lists all badges (locked & unlocked)
   ├── POST /api/achievements/check → evaluates conditions and unlocks new badges
   └── MongoDB collection: "user_achievements"

3. SUBJECTS API
   ├── File: backend/app/api/subjects.py             [NEW]
   ├── GET    /api/subjects      → lists user's subjects
   ├── POST   /api/subjects      → creates a new subject
   ├── DELETE /api/subjects/{id} → deletes a subject
   └── MongoDB collection: "subjects"

4. LEADERBOARD API
   ├── File: backend/app/api/leaderboard.py          [NEW]
   ├── GET /api/leaderboard/weekly → top users by sessions this week
   └── GET /api/leaderboard/typing → top users by best WPM all-time

5. UPDATED SESSIONS API
   ├── File: backend/app/api/sessions.py             [MODIFIED]
   └── Accepts optional subject_id when saving a session

6. WIRE ROUTERS INTO MAIN APP
   ├── File: backend/app/main.py                     [MODIFIED]
   └── Added goals, achievements, subjects, and leaderboard routers.
   └── Updated version to 3.0.0
