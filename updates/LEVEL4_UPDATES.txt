  SMART STUDY & PRODUCTIVITY TRACKER — LEVEL 4 UPDATES
  Social, Analytics & Smart Features
  Date: 2026-04-24
===================================================================

GOAL:
  Introduce social connectivity, detailed progress insights, and smart
  features. Added friend system, friend activity feed, notes, flashcards
  with spaced repetition (SM-2), 90-day heatmap analytics, personalized
  study tips, and push-notification-ready study reminders.

===================================================================
  BACKEND UPDATES
===================================================================

1. FRIENDS & ACTIVITY API
   ├── File: backend/app/api/friends.py              [NEW]
   ├── File: backend/app/api/activity.py             [NEW]
   ├── POST   /api/friends/request         → send friend request
   ├── GET    /api/friends/requests        → list pending incoming requests
   ├── PUT    /api/friends/accept/{req_id} → accept request
   ├── PUT    /api/friends/reject/{req_id} → reject request
   ├── GET    /api/friends                 → list all accepted friends
   ├── DELETE /api/friends/{friend_id}     → remove a friend
   ├── GET    /api/friends/search?q=       → search users by username
   ├── GET    /api/activity/friends        → friend study feed (last 24h)
   └── MongoDB collection: "friend_requests"

2. NOTES & FLASHCARDS API
   ├── File: backend/app/api/notes.py                [NEW]
   ├── File: backend/app/api/flashcards.py           [NEW]
   ├── CRUD endpoints for study notes (/api/notes)
   ├── CRUD endpoints for flashcards (/api/flashcards)
   ├── GET    /api/flashcards/review       → get cards due for spaced repetition
   ├── POST   /api/flashcards/{id}/review  → rate card 1-5 to update SM-2 schedule
   └── MongoDB collections: "notes", "flashcards"

3. ANALYTICS API
   ├── File: backend/app/api/analytics.py            [NEW]
   ├── GET /api/analytics/heatmap     → 90-day study heatmap data
   ├── GET /api/analytics/subjects    → time breakdown by subject
   ├── GET /api/analytics/trends      → weekly averages over 8 weeks
   └── GET /api/analytics/focus-hours → best focus hours of the day

4. SMART TIPS & REMINDERS API
   ├── File: backend/app/api/tips.py                 [NEW]
   ├── File: backend/app/api/reminders.py            [NEW]
   ├── GET /api/tips/daily      → personalized tip based on user stats
   ├── CRUD endpoints for study reminders (/api/reminders)
   └── MongoDB collection: "reminders"

5. WIRE ALL 7 NEW ROUTERS
   ├── File: backend/app/main.py                     [MODIFIED]
   └── Added friends, activity, notes, flashcards, analytics, tips, reminders routers.
   └── Updated version to 4.0.0 (total 39 active routes)
