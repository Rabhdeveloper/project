GOAL:
  Take the application to the ultimate level by introducing AI-powered

  external calendar integrations, and robust offline data sync.

===================================================================
  BACKEND UPDATES
===================================================================

1. REAL-TIME STUDY ROOMS API (WebSocket)
   ├── File: backend/app/websockets.py               [NEW]
   ├── File: backend/app/api/rooms.py                [NEW]
   ├── POST /api/rooms           → create a multiplayer study room
   ├── GET  /api/rooms           → list active public rooms
   ├── GET  /api/rooms/{room_id} → get room details
   ├── WS   /ws/rooms/{room_id}  → real-time connection for timer sync & chat
   └── MongoDB collection: "rooms"

2. AI "STUDY BUDDY" API
   ├── File: backend/app/api/ai_tutor.py             [NEW]
   ├── POST /api/ai/ask      → contextual Q&A based on user's notes
   ├── POST /api/ai/generate → auto-generate flashcards from text
   └── POST /api/ai/summary  → generate key bullet points from a long note

3. EXTERNAL CALENDAR API
   ├── File: backend/app/api/calendar.py             [NEW]
   ├── POST /api/calendar/sync   → push study reminders to external calendar
   └── GET  /api/calendar/events → fetch schedule to suggest study blocks

4. WIRE ALL LEVEL 5 ROUTERS
   ├── File: backend/app/main.py                     [MODIFIED]
   ├── Added rooms, ai_tutor, and calendar routers.
   └── Updated version to 5.0.0
