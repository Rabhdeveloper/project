1. BACKEND — Goals & Streaks API  [NEW: app/api/goals.py]
   - GET  /api/goals          → get user's daily goal + current streak
   - PUT  /api/goals          → update daily goal target {daily_target: 4}
   - GET  /api/goals/today    → today's progress (sessions done vs target)
   - The streak auto-calculates: consecutive days the user met their goal

2. BACKEND — Goals Model  [NEW: app/models/goals.py]
   - UserGoal: {user_id, daily_target (default 4), created_at}
   - Streak is computed on-the-fly from study_sessions collection

3. BACKEND — Achievements API  [NEW: app/api/achievements.py]
   - GET  /api/achievements      → list all achievements with locked/unlocked
   - POST /api/achievements/check → re-evaluate and unlock any new badges
   - Achievement definitions (hardcoded list):
       🔥 First Flame      — Complete 1 study session

       🏆 Century Club     — Complete 100 study sessions
       ⚡ Speed Demon      — Reach 60+ WPM in typing
       🎯 Sharpshooter     — Reach 95%+ accuracy in typing
       📅 Week Warrior     — 7-day study streak
       🗓️  Month Master     — 30-day study streak
       ⌨️  Typing Veteran   — Complete 50 typing tests
       🌟 Perfectionist    — Score 100% accuracy in a typing test
       💎 Diamond Focus    — Study 10+ hours total

4. BACKEND — Achievements Model  [NEW: app/models/achievements.py]
