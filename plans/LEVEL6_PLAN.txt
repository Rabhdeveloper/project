2. FRONTEND — Smartwatch App (WearOS & WatchOS)  [NEW: watch/ directory]
   - Standalone Pomodoro timer synced with the user's account.
   - Haptic feedback for focus/break transitions.
   - Quick dictation for capturing stray thoughts into the Notes inbox.

 ═══════════════════════════════════════════════════
  SECTION C — ENTERPRISE & CLASSROOM MODE
 ═══════════════════════════════════════════════════

3. BACKEND — Organizations API  [NEW: app/api/organizations.py]
   - POST /api/orgs          → create a classroom/team
   - POST /api/orgs/invite   → invite students
   - GET  /api/orgs/stats    → aggregate progress for teachers

4. FRONTEND — Classroom Dashboard  [NEW: class_dashboard_screen.dart]
   - Teacher view: monitor student focus times and assign specific flashcard decks.
   - Student view: shared assignments and org-specific leaderboards.

 ═══════════════════════════════════════════════════

 ═══════════════════════════════════════════════════

5. FRONTEND — Native Desktop Apps (MacOS / Windows)  [MODIFY: linux/mac/win runners]
   - System-wide "Deep Work" mode: integrates with OS APIs to block distracting websites and mute notifications during a Pomodoro session.
   - Always-on-top mini timer widget.

 ═══════════════════════════════════════════════════
  SECTION E — OPEN DEVELOPER API
 ═══════════════════════════════════════════════════

6. BACKEND — Developer Platform  [NEW: app/api/developer.py]

   - Webhooks: trigger external services on timer completion or achievement unlock.
