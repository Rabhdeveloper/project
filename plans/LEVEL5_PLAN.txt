1. BACKEND — AI Tutor API  [NEW: app/api/ai_tutor.py]
   - POST /api/ai/ask      → ask a question contextually based on notes
   - POST /api/ai/generate → generate flashcards automatically from a note
   - POST /api/ai/summary  → summarize a long note into key bullet points
   - Integrates with external LLM API (e.g., OpenAI/Gemini)

2. FRONTEND — AI Assistant Screen  [NEW: ai_assistant_screen.dart]

   - Accessible via a new FAB on the Dashboard or a dedicated tab

 ═══════════════════════════════════════════════════
  SECTION B — REAL-TIME STUDY ROOMS
 ═══════════════════════════════════════════════════

3. BACKEND — WebSockets for Rooms  [NEW: app/api/rooms.py, app/websockets.py]
   - POST /api/rooms           → create a multiplayer study room
   - GET  /api/rooms           → list active public rooms / friend rooms
   - WS   /ws/rooms/{room_id}  → real-time connection for timer sync
   - Broadcasts timer start, pause, and user presence

4. BACKEND — Rooms Model  [NEW: app/models/rooms.py]
   - StudyRoom: {_id, host_id, name, is_private, current_status, participants}
   - RoomCreate, RoomResponse schemas

5. FRONTEND — Study Rooms Screen  [NEW: rooms_screen.dart]
   - Lobby to view active friend rooms or public rooms
   - Inside Room: Shared Pomodoro timer synced via WebSocket
   - Participant list showing live status (Focusing, on Break)
   - Integrated basic chat/emoji reactions for motivation
